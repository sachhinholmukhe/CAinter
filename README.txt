HOW TO USE LOCALLY
------------------
1. Unzip this folder.
2. KEEP ALL FILES IN ONE FOLDER (do not move individual files out).
3. Double-click index.html — it opens in your browser.
4. From the hub, tap any "Open" button to launch that companion.

HOW TO MAKE QR CODES (recommended workflow)
-------------------------------------------
1. Host the whole folder online — easy free options:
     - Netlify Drop:  https://app.netlify.com/drop   (drag the folder, get a link)
     - GitHub Pages:  push the folder to a repo and enable Pages
2. Open the hosted index.html.
3. In the "Base URL" box at the top, paste your host URL (e.g. https://your-site.netlify.app/).
   The hub remembers it locally — set it once.
4. On any card, hit "Copy link" — the full URL of that file is copied.
5. Paste it into any free QR generator (qr-code-generator.com, qrcode-monkey.com).
   Download the PNG. Place it next to that topic in your slides / poster / book.

NOTE ON STORAGE
---------------
Marks and progress (Revision Tracker, Language Bank stars) save to the device's
browser storage. They stay on that device only. Hosting the suite and sharing
the URL is the cleanest way to give each student their own persistent progress.

LANGUAGE BANK AUDIENCE
----------------------
The Audit Language Bank is written for every audit student — CA Inter, CA Final,
ACCA and beyond. The rest of the suite follows the CA Intermediate, Paper 5
(Group II) syllabus.
